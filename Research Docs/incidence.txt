Concrete incident examples in YOUR system
1️⃣ Beacon anomaly incident

What happens
Agent usually beacons every ~10 seconds
Suddenly stops for 30–60 seconds
Then comes back

Why it matters
Process may have been killed
Host went offline
Defender interference

Incident
“Agent heartbeat anomaly detected”

2️⃣ Persistence confirmation incident

What happens
Agent reports persistence installed
Or reconnects after reboot

Why it matters
System modification occurred
Persistence mechanism is active

Incident
“Persistence mechanism established on host”
This is a high-severity incident in real SOCs.

3️⃣ Task retry incident

What happens
Same task retried multiple times
ACK delayed or missing

Why it matters
Network instability
Endpoint interference
Partial blocking

Incident
“Command delivery reliability degradation”

4️⃣ Command behavior anomaly

What happens
Long-running command
Unexpected output size
Multiple failures in sequence

Why it matters
Command misuse
System instability
Defensive controls triggered

Incident
“Suspicious command execution behavior”

5️⃣ Dormant-to-active transition

What happens
Agent silent for long time
Suddenly resumes beaconing

Why it matters
Host reconnected
VM snapshot restored
Laptop powered on

Incident
“Previously dormant endpoint re-activated”
